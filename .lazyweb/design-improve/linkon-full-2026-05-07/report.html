Linkon 전체 시스템 디자인 개선

2026-05-07 · 코드 분석 기반 · 홈 / 로그인·회원가입 / 서비스 페이지 / 어드민

TL;DR — 핵심 개선

가장 큰 기회는 숨겨진 auth 패널 복원이었습니다 — 브랜드 콘텐츠가 DOM에 존재했지만 display: none으로 숨겨져 있었습니다. 이를 복원하면 로그인/회원가입 페이지가 일반적인 폼에서 브랜드 경험으로 바뀝니다. 두 번째로 큰 문제는 홈 페이지의 서비스 카드 시각적 위계로, Vion(이용 가능)과 출시 예정 카드가 동일한 시각적 무게를 갖고 있어 우선순위 신호가 모호했습니다.

1

Auth 패널 데스크톱에서 복원 ⭐

로그인·회원가입 페이지의 분할 레이아웃이 이제 ≥1100px에서 동작합니다. 왼쪽 패널에 Linkon 로고, 브랜드 문구, Vion/Rion/Taxon 로고 행이 표시됩니다. 배경도 브랜드 색상(#f8f6ff → #f6f3ea)에 맞게 조정했습니다.

서비스 카드 시각적 위계 — Live vs 출시 예정 ⭐

Vion 카드에 lp-service-card--live 클래스(teal 테두리 + 높은 그림자)를 적용하고, Rion·Taxon에 lp-service-card--soon(아이콘 50% 불투명도, CTA 영역 72% 불투명도)을 적용했습니다.

유지된 좋은 패턴

  • 디자인 토큰 시스템 (variables.css) — 시맨틱 명명, 8px 베이스
  • 서비스 색상 팔레트 — Vion 틸 / Rion 블루 / Taxon 크림슨, 구별 명확
  • 카드 테두리 반경 시스템 — 28-36px, 프리미엄하고 일관적
  • 모바일 햄버거 드로어 — aria-expanded, 오버레이 포함, 잘 구현됨
  • 출시 알림 탭 UI — 탭 전환 패턴 우수
  • 인증 폼 에러 처리 — 포괄적인 에러 메시지 맵
  • 애니메이션 변수 — ease-out/spring, 부드러운 호버 전환
